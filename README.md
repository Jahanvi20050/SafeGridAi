# SafeGridAi - Women Safety Route Planner

## Problem
Women often choose shortest routes, not safest routes.

## Solution
SafeGrid AI uses AI agents + crime intelligence + routing algorithms to suggest safer routes.

## Features
- Safe route planning
- Crime hotspot detection
- Nearby police station detection
- Emergency SOS support
- Safety recommendations

## Tech Stack
- Python
- Streamlit
- Google ADK
- MCP Server
- Antigravity
- NetworkX
- OSMnx
- GeoPandas
- Folium

## Agent Architecture
Route Agent → Crime Agent → Emergency Agent → Recommendation Agent

## Setup
1. Clone repo
2. Install requirements
3. Add API key in .env
4. Run streamlit app
